## One-way decision

##Two-way decision
相当于一个fork，使用if-then-else语句
```python

if x>2 :
    print('Bigger')
else:
    print('Smaller')
print('All done')
```

## return values
return是python中的一个关键字，用于将值返回给调用的函数，并将控件从被调用函数转移到调用函数

语法为 return [value]，fruitful function
```python
def greet(lang):
    if lang =='es'
        return 'Hola'
    elif lang =='fr'
        return 'Bonjour'
    else:
        return 'Hello'
print(greet('en'), 'Glenn')
print(greet('es'), 'Sally')
print(greet('fr'), 'Michael')
```
## Multiple Parameters/Arguments
1、我们可以在函数定义中定义多个参数

2、我们需要匹配数字和引数以及参数的顺序
```python
def addtwo(a,b):
    added = a + b
    return added#将计算后得到的a+b的值重新赋给addtwo

x = addtwo(3,5)
print(x)
```
